# Fetcher
A simple, writte in zsh , neofetch alternative

## How to use

simple 
1. clone the repo(ig u know how to do that)

### 2. Install python
#### for example, on Arch btw
```
sudo pacman -S python
```


### 3 . run it

```
python fetcher.py
```
